V6.0 TOPOLOGY & PARALLEL FORMULAS

High-Side (Control FET)

  • 承受全部的電壓與電流交疊開關損耗 (Hard Switching)。
  • 需提供對側 Body Diode 的反向恢復電荷 $Q_{rr}$。
  • 佔空比 $D = V_{out} / V_{in}$。

Low-Side (Sync FET)

  • 實現零電壓切換 (ZVS)，開關損耗近乎為零。
  • 主導損耗為導通與死區時間的體二極體損耗。
  • 佔空比 $D_{LS} = 1 - (V_{out} / V_{in})$。

★ 並聯操作 (Parallel $N_p$) 物理模型

當系統採用 $N_p$ 顆 MOSFET 並聯時，系統總等效參數變化如下，並依此計算總功耗再均分至單顆 FET 進行熱分析：

  • 總等效內阻下降：$R_{DS(on)(sys)} = R_{DS(on)} / N_p$
  • 總閘極電荷增加：$Q_{g(sys)} = Q_g \times N_p$
  • 總輸出電容增加：$C_{oss(sys)} = C_{oss} \times N_p$
  • 單顆分擔功耗：$P_{total(per)} = P_{total(sys)} / N_p$

1. 導通損耗 (系統總耗散)

$$I_{rms(sys)} = \sqrt{Duty \times \left( I_{out}^2 + \frac{\Delta I_L^2}{12} \right)}$$ $$P_{cond(sys)} = I_{rms(sys)}^2 \times \left(\frac{R_{DS(on)@25^\circ C}}{N_p}\right) \times [1 + \alpha \times (T_j - 25)]$$

2. 切換與 $Q_{rr}$ 損耗 (系統總耗散，HS為主)

$$P_{sw(sys)} = \frac{1}{2} \times V_{in} \times I_{out} \times (t_r + t_f) \times f_{sw}$$ $$P_{coss\_qrr(sys)} = \left(\frac{1}{2} (C_{oss} \times N_p) V_{in}^2 f_{sw}\right) + \left((Q_{rr} \times N_p) V_{in} f_{sw}\right)$$

3. 死區時間與閘極驅動 (系統總耗散)

$$P_{deadtime(sys)} = 2 \times V_f \times I_{out} \times t_{dead} \times f_{sw} \quad \text{(僅 LS 發生)}$$ $$P_{gate(sys)} = (Q_g \times N_p) \times V_{gs} \times f_{sw}$$

4. 單顆熱疊代網絡 (Per FET)

$$P_{total(per)} = \frac{P_{cond(sys)} + P_{sw(sys)} + P_{coss\_qrr(sys)} + P_{deadtime(sys)} + P_{gate(sys)}}{N_p}$$ $$T_c = T_a + (P_{total(per)} \times \theta_{CA})$$ $$T_j = T_c + (P_{total(per)} \times \theta_{JC})$$
